Commentary Tab

Read verse-by-verse commentary alongside the text.

Selecting a commentary

Switch to the Commentary tab in the Companion panel. Use the picker in the module bar to choose from your installed commentaries. Your selection is remembered across sessions.

Reading commentary

Commentary entries appear for the current chapter automatically. Each entry shows the verse reference it covers. When you select a verse in the main panel, the commentary panel scrolls to highlight the entry covering that verse.

Installing commentaries

Commentary modules use the .cmtx format for e-Sword or the MyBible .SQLite3 format with a commentaries table. Import e-Sword files using the Import button in the Library toolbar. See Installing Modules for details.
