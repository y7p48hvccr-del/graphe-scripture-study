AI Summaries

Generate intelligent chapter summaries using Claude or a local Ollama model.

Claude (Anthropic)

ScriptureStudy can use Anthropic's Claude AI to generate chapter summaries. To enable this:

  1. Obtain an API key from console.anthropic.com
  2. Open Settings → AI Engine — Claude
  3. Paste your API key into the field provided

Summaries are generated on demand and are charged against your Anthropic API account at standard rates.

Ollama (local)

If you prefer to keep your data entirely on your Mac, ScriptureStudy supports Ollama — a free tool that runs open-source AI models locally. To set it up:

  1. Install Ollama from ollama.com
  2. Pull a model — ollama pull llama3 is a good starting point
  3. In Settings → AI Engine — Ollama, enter your model name and server URL (default: http://localhost:11434)

Ollama runs entirely offline — no data leaves your Mac.

Auto-generate summaries

Enable Settings → Behaviour → Auto-generate summaries to generate a summary automatically each time you open a new chapter. With this off, summaries are only generated when you explicitly request one.
