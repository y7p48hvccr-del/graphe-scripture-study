Web Tab

Open Bible reference websites alongside your reading.

Available websites

The Web tab provides quick access to a curated list of Bible reference websites. Use the picker in the module bar to switch between them. The site loads the passage currently selected in the main Bible panel automatically.

Verse-aware URLs

When you select a verse in the main panel, the web view updates to reflect that specific verse reference on supported sites.
