Writing Notes

Create and manage personal study notes linked to Bible verses.

Creating a note

Long-press any verse number in the Bible panel (hold for approximately 0.4 seconds). A new note is created and linked to that verse, and the Organizer tab opens automatically showing the new note ready to edit.

You can also create a free-standing note (not linked to a specific verse) by clicking the pencil icon in the Organizer sidebar header.

The note editor

The editor is a full-featured plain text environment built on macOS's native text system. It supports:

  • Full undo and redo history
  • macOS spell checking and autocorrect
  • The system Edit menu — Find & Replace, spelling, substitutions
  • Paste from any source including web pages, PDFs and other applications

Note menu commands

A Note menu appears in the menu bar when the editor is active, providing formatting shortcuts:

CommandShortcutEffect
Bold⌘BWraps selected text in **bold** markers
Italic⌘IWraps selected text in *italic* markers
Heading⌘⇧HInserts # at the start of the current line
Bullet List⌘⇧LInserts - at the start of the current line
Increase Font Size⌘+Increases the editor font size
Decrease Font Size⌘-Decreases the editor font size

Note titles

The note title appears at the top of the editor panel. Click it to edit the title directly. If a note is linked to a verse, the verse reference is used as the default title. The title is also used as the filename on disk.

Saving

Notes save automatically — you never need to press a save button. Changes are written to disk within one second of you stopping typing.

Where notes are stored

Notes are saved as plain text .txt files in ~/Documents/ScriptureStudy Notes/. Each file is human-readable and can be opened in any text editor. The first several lines contain metadata (title, book, chapter, verse, date, UUID) followed by a separator line and your note content.

Linked notes

A note linked to a specific verse appears in the Bible panel with a note icon next to that verse. Tapping the icon opens the note in the Companion panel's Notes tab. A note can be linked to multiple verses by creating separate notes for each, or to a chapter by leaving the verse field general.

Empty note deletion

If you create a note and leave it completely empty, it is automatically deleted when you navigate away from the Organizer tab. Notes with any content — even a single character — are always preserved.

Multiple notes per verse

You can create as many notes as you like for a single verse. Long-press the verse number again to add another. Each note is completely independent and editable separately in the Organizer.

Locking notes

Right-click a note in the sidebar to lock it. Locked notes cannot be edited and are protected from accidental changes. Click the lock icon in the note header to unlock.
