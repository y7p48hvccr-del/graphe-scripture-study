Module Library

Manage all your installed Bible study modules.

Sections

The Library organises your modules into four sections:

  • Bibles — all installed Bible translations. The active Bible shows a blue checkmark. Click Use on any Bible to make it active.
  • Commentaries — all installed commentary modules.
  • Dictionaries — all dictionary and Strong's lexicon modules. A purple Strong's badge shows the current Strong's selection; a green Dictionary badge shows the current plain dictionary selection. Use the Set as… menu to change either selection.
  • Other — devotionals, topical modules and anything else not recognised as a Bible, commentary or dictionary.

Module visibility

Click the eye icon on any module to show or hide it from the picker menus in the rest of the app. Hidden modules remain installed and can be unhidden at any time.

Modules folder

The current modules folder path appears at the bottom of the Library. Click Change to select a different folder.

Importing e-Sword modules

Click the Import e-Sword button in the Library toolbar. Select a compatible file and ScriptureStudy converts it to MyBible format and saves it into your modules folder. See Installing Modules for supported formats.
