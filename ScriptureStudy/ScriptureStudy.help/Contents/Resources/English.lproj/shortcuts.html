Keyboard Shortcuts

All keyboard shortcuts available in ScriptureStudy.

Navigation

ShortcutAction
⌘,Open Settings
⌘FFocus search (in Search tab)

Note editor

ShortcutAction
⌘BBold — wraps selection in **bold**
⌘IItalic — wraps selection in *italic*
⌘⇧HHeading — inserts # at line start
⌘⇧LBullet list — inserts - at line start
⌘+Increase font size
⌘-Decrease font size
⌘ZUndo
⌘⇧ZRedo
⌘ASelect all
⌘FFind in note

System Edit menu (active in note editor)

ShortcutAction
⌘⌥FFind and Replace
⌘;Check spelling
⌘:Show spelling and grammar panel
