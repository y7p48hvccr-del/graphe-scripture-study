AI Study Assistant

Claude and Ollama integration

Overview

ScriptureStudy can generate AI-powered study summaries and answer questions about the passage you're reading. Two AI backends are supported — Anthropic's Claude (via API key) and local Ollama models.

Setting up Claude

Go to Settings → AI → Claude API Key and enter your Anthropic API key. Once set, automatic chapter summaries appear at the top of each chapter as you read.

Setting up Ollama

Install Ollama on your Mac and pull a model (e.g. ollama pull llama3). In Settings, enter your Ollama server URL (default http://localhost:11434) and select your model. Ollama runs entirely locally with no API costs.

Study tab

The Study tab provides a chat interface for asking questions about the current passage. Type any question and the AI responds in context of the verses you're reading.

Auto summaries

When enabled in Settings, a brief summary is generated automatically when you load a new chapter. Chapter 1 of any book generates a full book overview alongside the chapter summary.
