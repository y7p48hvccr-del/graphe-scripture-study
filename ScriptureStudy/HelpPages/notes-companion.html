Notes — Companion Panel

Read and search your notes alongside the Bible text

Opening notes for a verse

When a verse has notes linked to it, a small note icon appears to the right of the verse text in the Bible panel. Tapping this icon switches the Companion Panel to the Notes tab and displays all notes linked to that verse.

Tapping a verse number opens the Commentary tab, not Notes. The note icon to the right of the verse text is what opens Notes.

All notes view

When no specific verse note is open, the Notes tab shows all your notes sorted by most recently updated. A search field at the top filters by title, content, or verse reference.

Note actions

Each note row shows a pencil icon to open it in the full Organizer editor, and a trash icon to delete it immediately. Right-click any note for a context menu with the same options.

Creating notes

Long-press a verse number in the Bible panel to create a new note linked to that verse. The note opens in the Organizer tab for full editing. You can also create notes directly from the Organizer tab.

Note content

Notes support basic Markdown formatting — use ⌘B for bold, ⌘I for italic, ⌘⇧H for a heading, and ⌘⇧L for a bullet list. Notes sync to iCloud automatically.
