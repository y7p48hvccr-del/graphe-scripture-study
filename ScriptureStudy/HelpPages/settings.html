Settings

All preferences in one place

Appearance

  • Theme — Light, Sepia, Blush, Light Grey or Charcoal
  • Accent colour — colour for verse numbers, icons and highlights
  • Font — reading font selection and size
  • Texture — optional paper grain overlay
  • Decoration — optional background decoration (currently disabled)

Behaviour

  • Auto summaries — generate AI chapter summaries automatically
  • Launch animation — show the quill animation on startup
  • Flash Strong's — highlight matching Strong's words when tapping

AI

  • Claude API key — your Anthropic API key for cloud AI
  • Ollama model — local AI model name
  • Ollama URL — server address (default http://localhost:11434)
