Bible Maps

Historical maps and geographic place lookup

Overview

The Maps tab provides a Bible atlas with 15 historical maps covering the Old and New Testaments, plus an index of 358 named biblical places. Each place links to the maps it appears on, and each map lists the key locations with relevant scripture references.

Installing the Maps module

The Maps tab requires BMaps.dictionary.SQLite3 in your modules folder — the same folder selected in the Library tab. Once the file is there, the Maps tab loads automatically on the next launch.

Browsing maps

The default view lists all 15 maps in order. Tap any map to open it. The map image loads at the top, followed by a list of named locations on that map. Tap any location to expand its verse references. Tap a verse reference to jump directly to that passage in the Bible panel.

Searching places

Tap the magnifying glass in the Maps tab header to search by place name. Type any biblical place — Jerusalem, Capernaum, Damascus — and matching entries appear with the maps they appear on and their associated verse references.

Verse context

When you tap a verse in the Bible panel, the Maps tab quietly checks whether any mapped places are associated with that verse. If it finds any, a badge appears at the top of the Maps tab showing how many places are nearby. Tapping the badge shows the relevant places without leaving the tab.

Map navigation

Each map has previous and next arrows for moving between maps in sequence. Tapping these navigates within the Maps tab itself without leaving the Companion Panel.

Verse references shown under map locations are tappable — they open that passage directly in the main Bible panel.
