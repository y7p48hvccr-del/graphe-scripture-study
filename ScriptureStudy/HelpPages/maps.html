Bible Maps

Historical maps and geographic place lookup

Overview

The Maps tab provides a Bible atlas and place lookup backed by a locally installed atlas module. The app reads the atlas from your modules folder rather than bundling a map database inside the application.

Installing an atlas module

Place a compatible SQLite atlas module in your modules folder — the same folder selected in the Library tab. The Maps tab will attempt to load atlas modules with names such as BMaps, BibleMaps, BibleAtlas, or Atlas.

Testing note

Atlas content is supplied by the module you install, so licensing and attribution requirements depend on that source. Confirm you have permission to distribute any atlas module beyond private testing.
