Library

Managing your Bible modules

Modules folder

ScriptureStudy reads all MyBible-format SQLite files from a single folder. Click Choose Folder in the toolbar to set or change this folder. The folder path is shown at the bottom of the module list. Click Change to pick a different folder.

Refreshing

Click the ↺ Refresh toolbar button to rescan the modules folder after adding or removing files.

Search

A search box at the top of the module list filters modules by name, language, author and other metadata stored in each module's info table. Common abbreviations are recognised — searching KJV finds "King James Version", LXX finds Septuagint modules, TSK finds Treasury of Scripture Knowledge, and so on.

Favourites

The Favourites section at the top of the list shows all currently visible modules across every category. Each entry shows the module name and a colour-coded type badge. Modules are added to Favourites automatically when their visibility is turned on, and removed when turned off.

Categories

Modules are grouped into sections in alphabetical order:

  • Bibles — Bible translation modules
  • Commentaries — verse and chapter commentary
  • Cross-References — OBX, TSK and footnote cross-reference systems
  • Devotionals — daily devotional modules
  • Dictionaries — word definition dictionaries
  • Lexicons — Strong's Hebrew and Greek lexicons
  • Other — unrecognised module types
  • Reading Plans — day-by-day Bible reading schedules

Visibility toggle

Each module has an eye icon on the right. Click it to hide or show the module in the various pickers throughout the app. Hidden modules remain installed — they just don't appear in dropdowns. Use this to keep your pickers tidy when you have many modules installed.

Hide All / Show All

Each section header has a Hide All / Show All button. This toggles the visibility of every module in that section at once.

Setting active modules

For Bibles, click Use to set a module as the active Bible. For dictionaries, lexicons and cross-reference modules, click Set as… to assign a module to its role. Only one module of each type can be active at a time.

Importing e-Sword modules

Click the Import e-Sword toolbar button to convert an e-Sword or MySword module file into MyBible format. The converted file is saved directly to your modules folder and scanned automatically.
