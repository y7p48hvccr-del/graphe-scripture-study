Strong's Numbers

Hebrew and Greek word lookup

Overview

Some Bible modules include Strong's tagging — each word is linked to a number in Strong's Hebrew (Old Testament) or Greek (New Testament) lexicon. When a Strong's-tagged Bible is active, individual words in the text are tappable.

Looking up a word

Tap any word in the Bible text. If that word has a Strong's number, the companion panel switches to the Dictionaries tab and displays the lexicon entry — the original word, its transliteration, pronunciation, definition and usage notes.

Selecting a Strong's lexicon

Go to the Library tab and find your Strong's lexicon module in the Lexicons section. Click Set as… → Default Strong's module. Only one lexicon can be active at a time, but you can switch between Hebrew and Greek lexicons if you have both installed.

Flash highlighting

When Strong's flash mode is enabled in Settings, tapping a word briefly highlights all words sharing the same Strong's number in the current chapter. This helps you see where the same original word is used across a passage.

Context menu

Right-clicking any word in a Strong's-tagged Bible gives options to copy the word, look it up in the system dictionary, or show its Strong's entry.
