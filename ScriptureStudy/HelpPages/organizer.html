Organizer

Calendar, reading plans, prayer and personal notes

Layout

The Organizer tab is divided into two panels. The left panel contains the calendar at the top and a scrollable bookmarks list below. The right panel contains the reading plan card, a daily summary of prayer and reminder items, and the note editor.

Calendar

The calendar occupies the top half of the left panel. It shows the current month with Monday as the first day of the week. Navigation arrows move between months and a Today button returns to the current date.

Each day cell can show coloured indicator dots:

Dot colourMeaning
BlueA reading plan passage is scheduled for this day
PurplePrayer requests or reminders are scheduled for this day
GreenThis day has been marked as complete

Tap a day to select it and update the right panel to show that day's content. Long-press a day to open the event popover.

Prayer and reminder popover

Long-pressing a calendar day opens a popover showing all prayer requests and reminders scheduled for that day. From the popover you can:

  • View existing items with their type (🙏 prayer / ❗ reminder)
  • Mark items as done with the circle button
  • Delete items with the ✕ button
  • Add a new prayer request or reminder using the type buttons and text field
  • Set the repeat rule — Once, Weekly or Daily
Items set to Weekly repeat every week on the same day. Daily items appear on every day from their start date onwards.

Reading plan card

The reading plan card in the right panel shows the Bible passage assigned to the selected day. A picker lets you switch between installed reading plan modules. Tap Read → to navigate the main Bible panel to that passage.

Day summary

Below the reading plan card, a summary shows all prayer requests and reminders scheduled for the selected day. Tick buttons let you mark items as done directly from the summary. The + button opens the event popover for the selected day.

Note editor

The bottom of the right panel contains a full word processor. It displays the most recently created verse note. When you long-press a verse number in the Bible panel, the new note appears here automatically and the Organizer tab opens. Notes are saved automatically — switching to a different note saves the previous one immediately.

Bookmarks

The lower-left panel shows your bookmarked chapters in a scrollable list. Tap the arrow next to any bookmark to navigate the main Bible panel to that passage. Right-click a bookmark to delete it.
