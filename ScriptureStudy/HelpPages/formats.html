Supported File Formats

What module files ScriptureStudy can read

Native format — MyBible SQLite

ScriptureStudy reads MyBible-format SQLite files directly. These are standard SQLite3 databases with a .SQLite3 extension. The module type is detected automatically from the tables present:

TableModule type
versesBible
devotionsDevotional
reading_planReading Plan
cross_referencesCross-Reference (OBX)
commentaries + TSK/footnote flagCross-Reference (TSK)
commentariesCommentary
dictionary + is_strong=trueLexicon
dictionary + is_strong=falseDictionary

Imported formats — e-Sword / MySword

Use the Import e-Sword button in the Library toolbar to convert e-Sword (.bblx, .cmtx, .dctx) and MySword (.mybible) files into MyBible format. The converted file is saved to your modules folder automatically.
