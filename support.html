Getting Started

The app opens but shows no Bible text. What do I do?
Graphē One requires Bible modules to be downloaded separately. Go to the Archives tab and tap Choose Modules Folder to point the app to a folder containing MyBible-format modules. Free modules for hundreds of languages are available at mybible.zone.
Where do I get Bible modules?
Visit mybible.zone to download free Bible translations, commentaries, lexicons, devotionals and more. Download the files to a folder on your Mac, then point Graphē One to that folder via the Archives tab.
What module format does Graphē One use?
Graphē One reads MyBible-format SQLite modules (.sqlite3, .bible, .commentary, .dictionary and .graphe encrypted formats).
How do I set up the AI features?
The AI chat and chapter summary features require either a local Ollama installation or an Anthropic API key. Configure these in Settings → AI. These features are entirely optional — the app is fully functional without them.

Common Questions

The EPUB reader says "Coming Soon" — when will it be available?
EPUB support is coming in a future update. In the meantime, PDF books work beautifully. You can convert EPUB files to PDF using the free tool Calibre.
My maps aren't showing images. What's wrong?
Make sure your modules folder contains the BMaps module (BMaps.dictionary.graphe or similar). If it's there, try pressing the Refresh button in the Archives tab to re-scan the folder.
I changed my modules folder and now nothing loads.
Go to the Archives tab and tap Choose Modules Folder again to re-grant access to your folder. macOS security requires you to reconfirm folder access after certain changes.
What macOS version does Graphē One require?
Graphē One requires macOS 14 (Sonoma) or later.
